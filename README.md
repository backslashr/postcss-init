# PostCSS Init.

Boilerplate-ish setup to quickly spin-up a project that uses PostCSS.

- fork this repo.
- `mkdir style && cd`
- `touch index.css`
- `mkdir components`
- import components in index.css

Uses the following packages:
```
postcss-import
postcss-mixins
postcss-simple-vars
postcss-color-function
postcss-nested
postcss-extend
postcss-discard-comments
postcss-discard-empty
postcss-preset-env
```
